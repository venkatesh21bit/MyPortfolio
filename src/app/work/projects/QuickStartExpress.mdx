## Overview

Quick Start Express is a simple CLI tool to generate Express servers from multiple available templates. This npm package streamlines the process of setting up Express.js applications by providing pre-configured templates for different use cases.

## Installation & Usage

```bash
# Install globally
npm install -g quick_start_express

# Generate new project
quick_start_express my-app

# Choose from available templates
quick_start_express my-app --template api
```
