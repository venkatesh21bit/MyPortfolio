## Overview

ClarityLegal is an AI-powered legal document analysis platform that democratizes sophisticated contract review for businesses of all sizes. Upload a legal document and get instant AI-powered insights on risks, obligations, and key terms in plain English, powered by Google's Vertex AI.

## Technical Architecture

### Frontend Stack
- **Framework**: Next.js 14 with App Router and TypeScript
- **Styling**: Tailwind CSS for responsive design
- **PDF Viewer**: Adobe PDF Embed API
- **PDF Processing**: PDF.js for text extraction
- **State Management**: React Hooks (useState, useEffect, useContext)

### Backend Stack
- **Framework**: FastAPI (Python 3.11+)
- **Authentication**: OAuth 2.0 + JWT tokens
- **API Documentation**: OpenAPI (Swagger) auto-generated

### Google Cloud Platform Integration
- **Compute**: Cloud Run for serverless container deployment
- **Storage**: Cloud Storage for document repository
- **Database**: Firestore NoSQL database
- **AI/ML**: Vertex AI for embeddings, NLP, and generative AI
- **Document AI**: OCR and text extraction
- **Secrets**: Secret Manager for API key management
- **Monitoring**: Cloud Monitoring for performance tracking

### AI & Machine Learning
- **Embeddings**: Vertex AI text-embedding-gecko model
- **Search**: RAG (Retrieval-Augmented Generation)
- **NLP**: Natural Language API for entity extraction
- **Generative AI**: GPT-4 for summarization and chat
- **Vector Search**: Semantic similarity matching

## System Workflows

### Document Upload Flow
```
User uploads PDF ‚Üí Frontend validates ‚Üí Backend receives ‚Üí 
Cloud Storage saves ‚Üí Document AI extracts text ‚Üí 
Vertex AI generates embeddings ‚Üí Firestore stores metadata ‚Üí
Success response to frontend
```

### RAG Search Flow
```
User selects text ‚Üí Frontend sends query ‚Üí Backend tokenizes ‚Üí
Vertex AI creates query embedding ‚Üí Vector similarity search ‚Üí
Retrieve top-k matches ‚Üí Rank by relevance ‚Üí Return snippets ‚Üí
Frontend displays with source attribution
```

### Risk Detection Flow
```
Document uploaded ‚Üí Extract clauses ‚Üí NLP entity recognition ‚Üí
Pattern matching against risk database ‚Üí ML risk classification ‚Üí
Severity scoring ‚Üí Group by category ‚Üí Return structured risks ‚Üí
Frontend displays with color coding
```
